
# Car Washing System - Mini Project

This repository contains a simple **car washing system simulation**.  
It demonstrates how a structured workflow can be implemented to handle customer requests, washing stages, and completion tracking.

## Features
- Customer request handling
- Multiple washing stages (pre-wash, soap, rinse, dry)
- Status updates for each stage
- Reporting of completed washes

## Project Files
- `src/` → Contains the main program and supporting modules
- `docs/` → Includes diagrams or notes explaining the workflow
- `README.md` → Overview of the project

## Tech Stack
- Assembly language (MCU 8051 IDE)
- 8051 Micro Controller

## Outcome
The project successfully simulates the process of a car wash, showing how tasks can be automated and tracked step by step.
